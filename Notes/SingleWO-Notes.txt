In the single workorder export, the important data lives in cells: V4 - Incident A8 - customer id C8 - first name J8 - last name AC10 - Facility AE8 - Room# D14 - specific request D15 - product type D16 - product brand D17 - Model Family K26 - Description Also I found AB15 - Facility & Room #, this is where a user can manually assign a room number for this work order if they're putting a request in for somebody else. This might be more useful than the AE8 one since that just grabs the room number from the users profile, but AB15 is also an optional field when submitting a workorder so not all will have this filled in.
